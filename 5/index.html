kesimpulan atau hasil pertemuan 5 yang saya ambil adalah:

  1. Cara Instal Tema:
  2. 1. Langsung online https://wordpress.org/themes/
  3. 2. Semi Online. Unggah file zip tema
  4. 3. Pindahkan folder tema
  5. klik tampilan. pilih tema terus klik pojok atas tambah baru dan pilih tema yang di inginkan klik instal
  6. Cara install plugin classic widget:
  7. - Pilih akses tampilan -> Widget.
  8. - Kemudian download classic widget.
  9. - Taruh file download di folder mana saja, gak perlu di extract, lokasinya diingat.
  10. - Klik menu Plugin -> Tambah baru.
  11. - Kemudian klik Unggah Plugin, nah pilih file zip yang kita download tadi.
  12. - Klik Instal sekarang dan aktifkan
