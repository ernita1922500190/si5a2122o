kesimpulan atau hasil pertemuan 5 yang saya ambil adalah:

  1. Cara Instal Tema:
  2. - Langsung online ->
  3. - Pastikan memilih menu utama horizontal, kemudian klik Pilih
  4. - Jika mau tambah menu tinggal pilih saja dan klik tambahkan ke menu dan atur menu nya sesuai yang diinginkan
  5. Cara install plugin classic widget:
  6. - Pilih akses tampilan -> Widget.
  7. - Kemudian download classic widget.
  8. - Taruh file download di folder mana saja, gak perlu di extract, lokasinya diingat.
  9. - Klik menu Plugin -> Tambah baru.
  10. - Kemudian klik Unggah Plugin, nah pilih file zip yang kita download tadi.
  11. - Klik Instal sekarang dan aktifkan
